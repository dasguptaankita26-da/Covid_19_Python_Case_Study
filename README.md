# COVID-19 Data Analysis using Python

## Project Overview

This project focuses on analyzing COVID-19 data using Python to understand the spread of the pandemic, compare country-wise trends, evaluate recovery and death rates, and generate meaningful insights through data visualization.

The analysis was performed using real-world COVID-19 datasets and involved data cleaning, transformation, exploratory data analysis (EDA), and visualization.

---

## Objectives

* Analyze COVID-19 confirmed, recovered, and death cases.
* Compare infection trends across different countries.
* Calculate and visualize death rates.
* Identify pandemic waves and peak periods.
* Understand recovery patterns and country-wise impact.
* Generate insights through statistical analysis and visualizations.

---

### Data Cleaning Performed

### 2. Death Rate Analysis

Calculated the COVID-19 death rate using:

#### Visualization

## Key Findings

* Multiple COVID-19 waves were observed between 2020 and 2021.
* France experienced higher infection peaks compared to Germany and Italy.
* Death rates varied significantly across provinces and regions.
* Recovery trends improved over time due to healthcare interventions and vaccination efforts.
* Understanding cumulative data was critical before performing meaningful analysis.

---

## Challenges Faced

One of the major challenges was working with cumulative COVID-19 data.

Since cumulative figures continuously increase over time, the data required careful interpretation and transformation before conducting trend analysis. Additional effort was spent understanding the dataset structure and converting it into a format suitable for generating actionable insights.
